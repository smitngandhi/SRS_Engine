# SpecForge AI Worker

This is the background processing engine for SpecForge AI, hosted on Hugging Face Spaces for high-performance SRS generation.

- **Frontend**: Hosted on Render
- **Queue**: Upstash Redis
- **Database**: MongoDB Atlas
- **RAM**: 16GB (HF Free Tier)
